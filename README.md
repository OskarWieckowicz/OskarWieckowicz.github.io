# OskarWieckowicz.github.io

Personal website for Oskar Wieckowicz.

This repository contains a lightweight static site deployed to GitHub Pages. It serves as a simple personal landing page with profile information, project links, and basic SEO metadata.

## Stack

- HTML
- CSS
- GitHub Pages
- GitHub Actions

## Deployment

The site is deployed through GitHub Pages using the workflow in [`.github/workflows/deploy.yml`](./.github/workflows/deploy.yml).

To publish changes:

1. Commit your updates to `main`.
2. Push to GitHub.
3. Ensure GitHub Pages is configured to use `GitHub Actions` as the source.

After the workflow completes, the site is available at:

`https://oskarwieckowicz.github.io/`
